# Calisthenics Coach Review, June 21 2026

## Bottom line
This plan has a solid weekly skeleton, but it is **not yet fully what a good calisthenics coach would build for Moshe's current request**.

The big reason is simple: the source plan still assumes some external loading or household setup tricks, while the current direction is moving toward a cleaner **bodyweight-first, low-friction, at-home plan**.

## What is good already
- 4 training days per week is reasonable.
- 2 run days per week is realistic and useful.
- 2 rest days per week is healthy and sustainable.
- Strength sessions staying around 20 to 30 minutes is a strong constraint.
- Week 4 as a consolidation week is a good coaching instinct.
- Push, legs, trunk, and running all get repeated exposure.

## Where it is not aligned yet

### 1. It is not truly equipment-light in the source of truth
The written plan still relies on:
- backpack rows
- one-arm backpack rows
- feet-elevated / decline push-up setup
- Bulgarian split squat setup
- optional armor-vest loading

That means the bot can hide some drills, but the underlying plan is still not fully aligned with a pure low-friction calisthenics standard.

### 2. Pulling is the weakest programming area
The plan assumes loaded rows as the main pull solution.

That is understandable, but from a coaching perspective it creates a gap:
- if Moshe wants true calisthenics style progression, pulling should either have a real progression path
- or the plan should openly become a temporary push-legs-core-running block until a bar / rings / anchor setup exists

Right now it sits in between.

### 3. The benchmark logic is only partly individualized
The plan uses push testing and then assumes pull is roughly on par.
That is workable for a stopgap plan, but it is not how a careful coach would want to prescribe pull progression for several weeks.

### 4. There is no explicit restart / re-entry rule
If Moshe misses 2 to 4 weeks, the plan should say exactly what happens next.
A coach-built plan should include a short re-entry rule such as:
- miss 3 to 6 days: resume current week
- miss 1 to 2 weeks: repeat previous week at 80 to 90 percent volume
- miss 3 or more weeks: restart from a fresh week-1 calibration

### 5. The progression is decent but not yet tightly coached
The structure progresses, but it would be stronger with clearer coaching rules around:
- reps in reserve
- when to upgrade an exercise variation
- when to add volume versus when to hold steady
- what counts as too easy / too hard

## What a better V2 should probably look like

### Option A, true no-equipment calisthenics block
Build the next version around movements that need almost nothing:
- push-up family
- pike push-up family
- split squat / reverse lunge family
- hamstring walkout / bridge family
- hollow / dead bug / side plank family
- running

And for pull, use one honest rule:
- either add a minimal towel / table / door-anchor solution with explicit safety rules
- or temporarily remove fake pull strength claims and focus on posture / scap / trunk accessories until real pulling equipment exists

### Option B, minimal-equipment calisthenics block
If a backpack, chair, and maybe one anchor point are acceptable, then say that clearly and build the plan honestly around that.
That version can be strong, but it should stop pretending to be no-equipment.

## Recommendation
The next rewrite should choose **one clear identity**:
1. **strict no-equipment bodyweight plan**, or
2. **minimal-equipment home calisthenics plan**.

Right now the plan mixes both, and that is the main mismatch.
